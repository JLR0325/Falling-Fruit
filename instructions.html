Falling Fruit Sorter

Game Instructions

Game Goal

Items fall from the sky and the player moves a basket left and right to catch the correct items. Catching the correct items increases the score. Catching the wrong items reduces the score or lives.

How to Play

  1. Enter your player name on the launcher page.
  2. Select the difficulty level.
  3. Select the game length.
  4. Choose a sort mode.
  5. Select optional features such as double points or random sizes.
  6. Click Open Game Window.
  7. In the game window, click Start Game.
  8. Use the Left Arrow and Right Arrow keys to move the basket.
  9. Catch only the correct items for the current rule shown on screen.

Sort Modes

  • Catch Fruit Only: catch fruit, avoid non-fruit items
  • Catch Healthy Food Only: catch healthy items, avoid junk food
  • Catch One Colour Group: catch only items of the target colour group

Difficulty Levels

  • Easy: wider basket, slower falling speed, fewer items
  • Medium: medium basket width and medium speed
  • Hard: narrower basket, faster speed, more items

Buttons in the Game Window

  • Start Game starts a new game
  • Pause / Resume pauses or resumes the game
  • Save Session stores the current game state
  • Load Session restores a saved game
  • Reset Game clears the current game
  • Back to Settings closes the game window and returns to the launcher

Scoring

  • Catching correct items adds points
  • Catching incorrect items reduces points and may reduce lives
  • Missing correct items may reduce lives
  • If Double points is enabled, correct catches score double
